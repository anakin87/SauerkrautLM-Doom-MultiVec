# ASCII Converter API

API reference for the DOOM frame-to-ASCII conversion pipeline.

---

## Converter

The `AsciiConverter` class handles the conversion of VizDoom screen, depth, and labels buffers into ASCII text representations suitable for tokenization and model encoding.

::: doom_multivec.ascii.converter

---

## Character Set

Constants defining the ASCII character sets used for brightness mapping and entity identification.
